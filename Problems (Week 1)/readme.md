# Problem sets for the first week

## FizzBuzzNoLoop
### Problem Statement
| Expected Runtime  | Memory Limit  |
| ----------------  | ------------- |
|       |   |
#### Description
#### Input Format
  A single integer (n) as described in the problem description.
#### Output Format
  Print either:
  * 'Fizz' if n is divisible by 3,
  * 'Buzz' if n is divisible by 5,
  * 'FizzBuzz' if n is divisible by 3 and 5,
  * else n if neither.
#### Example
* Input 0
  <pre>
    1
  </pre>
* Output 0
  <pre>
    1
  </pre>
* Input 1
  <pre>
    3
  </pre>
* Output 1
  <pre>
    Fizz
  </pre>
* Input 2
  <pre>
    15
  </pre>
* Output 2
  <pre>
    FizzBuzz
  </pre>
#### Constraints
0 < n < 2,147,483,647
